# votebox

сервис опросов на Go, на чистой стандартной библиотеке

цель проекта: показать конкурентную модель Go на практике: горутины, каналы, `select`, `sync.RWMutex` и graceful shutdown через `context`

## идея

голоса не записываются в хранилище напрямую из HTTP-обработчика, обработчик кладёт голос в канал и сразу отвечает клиенту `202 Accepted`
фоновая горутина-воркер вычитывает голоса из канала по одному и применяет их к хранилищу, запись идёт асинхронно через очередь
фронт на Bootstrap, API через fetch

## запуск

```bash
go run .
```

# данные

данные хранятся в оперативной памяти (`map` внутри `Store`)
хранилище инкапсулировано методами `Store`, поэтому переход на постоянную БД потребует изменить тела этих методов, не затрагивая HTTP-слой и воркер
